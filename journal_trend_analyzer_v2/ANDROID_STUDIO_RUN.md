## Android Studio

1. **Run → Edit Configurations**
2. **XÓA** mọi config loại `Dart Command Line Application`
3. Giữ/chọn config **Flutter** tên `main.dart`
4. Toolbar: `[ main.dart (Flutter) ]` + device → Run ▶
